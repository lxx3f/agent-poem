# 数据库设计
## 1. writer（作者表）
| 字段名   | 类型         | 说明         |
|----------|--------------|--------------|
| id       | INT(11)      | 主键，自增   |
| name     | VARCHAR(50)  | 作者姓名     |
| dynasty  | VARCHAR(20)  | 朝代         |
| intro    | TEXT         | 作者简介     |
| create_at| DATETIME     | 创建时间     |

## 2. poetry（诗词表）
| 字段名   | 类型         | 说明                  |
|----------|--------------|-----------------------|
| id       | INT(11)      | 主键，自增            |
| title    | VARCHAR(100) | 诗词标题              |
| content  | TEXT         | 诗词内容              |
| dynasty  | VARCHAR(20)  | 朝代                  |
| writer_id| INT(11)      | 外键，关联writer.id   |
| keywords | VARCHAR(200) | 关键词（逗号分隔）    |
| milvus_id| VARCHAR(50)  | 关联Milvus向量ID      |
| create_at| DATETIME     | 创建时间              |

## 3. sentence（诗句表）
| 字段名   | 类型         | 说明                  |
|----------|--------------|-----------------------|
| id       | INT(11)      | 主键，自增            |
| content  | VARCHAR(200) | 诗句内容              |
| poetry_id| INT(11)      | 外键，关联poetry.id   |
| create_at| DATETIME     | 创建时间              |
