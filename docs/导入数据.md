# 诗云（Shiyun）项目

## 数据导入操作说明文档

---

## 1. 数据来源说明

### 1.1 数据集简介

数据来源：`chinese-gushiwen`

包含三类核心数据：

| 类型 | 数量     | 说明       |
| -- | ------ | -------- |
| 古文 | ~10000 | 诗、词、歌、赋等 |
| 作者 | ~3983  | 含简介      |
| 名句 | ~10000 | 流传名句     |

---

### 1.2 数据文件结构

```text
data/
├── guwen/          # 古文（10 个 JSON 文件，每个 1000 条）
├── writer/         # 作者（4 个 JSON 文件）
└── sentence/
    └── sentences.json   # 名句（NDJSON 格式）
```

**重要说明**

* 文件格式为 **JSON Lines（一行一个 JSON 对象）**

---

## 2. 系统环境要求

### 2.1 基础环境

| 组件                      | 版本建议  |
| ----------------------- | ----- |
| Python                  | ≥ 3.9 |
| MySQL                   | ≥ 8.0 |
| Milvus                  | ≥ 2.3 |
| Docker / Docker Compose | 最新稳定版 |

---

### 2.2 本地 Embedding 服务（Ollama）

```bash
ollama pull nomic-embed-text
```

默认地址：

```
http://localhost:11434
```

---


## 3. 导入脚本说明

导入脚本位置：

```text
backend/scripts/import_data.py
```

主入口：

```bash
python import_data.py
```

## 4. 推荐执行流程（一次完整导入）

```bash
# 1. 启动基础服务
docker compose up -d

# 2. 执行导入
python backend/scripts/import_data.py
```
